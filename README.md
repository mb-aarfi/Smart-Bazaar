# Smart Bazaar - Empowering Retail with Intelligence

**Smart Bazaar** is a cross-platform mobile and web application developed using **React Native** and **Python**, designed to help retailers optimize their business. By integrating machine learning models, Smart Bazaar enables retailers to analyze business performance, forecast demand, and manage real-time stock updates‚Äîempowering smarter business decisions and boosting profitability.

## üåü Idea & Approach

Retailers face challenges in managing their businesses effectively, from tracking inventory to predicting demand accurately. **Smart Bazaar** bridges this gap with an innovative solution, offering a mobile web app that brings together **data-driven insights** and **real-time stock management** to give retailers the tools they need to stay ahead in a competitive market.

### Key Features:
1. **Performance Analytics**: Leverage machine learning to analyze sales data and provide actionable insights on business performance.

5. **Cross-Platform Support**: Accessible on both mobile and web, enabling seamless usability across all devices.

## üîß Tech Stack

- **Frontend**:
  - **React Native**: Cross-platform mobile app development for iOS and Android.
  - **HTML5, CSS3**: Responsive design for the web version of the app.
  - **JavaScript (React)**: Building dynamic UI components and interactions.
  
- **Backend**:
  - **Python (Flask)**: Backend logic and API handling.

  - **Scikit-Learn**: Implementing machine learning models for demand forecasting and sales prediction.

- **Database**:
  - **Firebase**: Real-time database integration for stock updates and user management.
  - **MongoDB**: Non-relational database for storing sales, stock, and user data.

- **Cloud**:
  - **Google Cloud**: Hosting and scaling backend services.
  - **Heroku**: Quick deployment and hosting for the backend.

## üìö How to Install & Run

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/SmartBazaar.git
   ```

2. Install dependencies for the React Native app:
   ```bash
   cd SmartBazaar
   npm install
   ```

3. Start the React Native server:
   ```bash
   npm start
   ```

4. Navigate to the `backend` folder to install Python dependencies:
   ```bash
   cd backend
   pip install -r requirements.txt
   ```

5. Run the Flask server:
   ```bash
   python app.py
   ```

6. Set up Firebase and MongoDB by configuring the necessary environment variables as outlined in the `config/` directory.
